Class: Slider

Slider

Responsive slider with parallax capabilities

Constructor

new Slider(selector, optionsopt)

Parameters:
Name Type Attributes Default Description
selector String class or ID to select the rails of the slider.
options Object <optional>
 {} Options for configuring instance of the class.
Properties
Name Type Description
amountOfSlides Number Amount of slides per time in view.
autoplay Boolean True or False to autoplay.
autoplaySpeed Number Time between slides in ms (milliseconds).
activeSlideClass String Class that is added to the slides in view.
dots Boolean If true outputs a unordered list with dots for navigation.
easing String String with easing given. Can be any CSS easing available.
forceSlideWidth Boolean Forcefully set the width calculated on the width of the slider.
prevButton HTMLElement Select an HTMLElement for navigating back.
nextButton HTMLElement Select and HTMLElement for navigating forward.
pauseOnHover Boolean True to pause on hovering over the slider. Resets the autoplay time on mouseout.
parallax String | Boolean Creates a parallax effect while sliding. String with class, id or element to select the parallax layer within the slide. True to use the first child of the slider as parallax layer.
parallaxIntensity Number The normal speed divided by the intensity. Eg: 2 = normalSpeed / 2 = 50%. The closer to 0 the more intense the effect.
slidesToScroll Number The amount of slides that have to be moved when sliding.
scroll Boolean True to scroll with mousewheel.
scrollIntensity Number
touch Boolean Set true if slider has to be swipeable on touch devices.
touchTreshold Number Amount of pixels that has to be swiped to move to the next slide.
transitionSpeed Number Duration of transition in ms (milliseconds).
responsive Array.<Array> Array with object that contains a breakpoint and an object with settings like the options paramter of the class.
Properties
Name Type Description
breakpoint Number Number in px to set the breakpoint.
options Object Options like the options parameter that override the initial settings given when the breakpoint is met.
