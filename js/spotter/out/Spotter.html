Class: Spotter

Spotter

Check if selected element(s) are in the view of the window. Animate them in or add your own function to deal with the results.

Constructor

new Spotter(selector, optionsopt)

Creates an instance of the Spotter class. Takes 1 mandatory argument and 2 optional arguments.
Parameters:
Name Type Attributes Default Description
selector string | HTMLElement | Array.<HTMLCollection> Name of elements in string or HTMLCollection.
options Object <optional>
 {} Options for configuring the class.
Properties
Name Type Description
animation Object | String | Boolean Object with animation properties or String with classname or Boolean with false for no animation.
Properties
Name Type Description
name String Name of keyframe animation.
duration Number Duraction in ms (milliseconds).
fillMode String Fillmode of animation.
delay Number Delay of animation in ms (milliseconds).
timingFunction String Easing of animation.
iterationCount String Amount of animation iterations.
directions String Direction of animation.
playState String Play-state of animation.
delay Number Delay of firing animation and callback in ms (milliseconds).
stagger Number Amount of delay between elements in ms (milliseconds).
offset Number Start counting from given offset.
restore Boolean Restore to first position when element leaves view.
partInView Boolean | Number True: fires when element is partialy in view, False: fires when element is fully in view, Number = fires when element passes window width and height divided by the number given.
lazyLoadImages Boolean Load images when they enter into view. Image can be the selector or children of the selector. Images have to have an attribute of 'data-src' with the URL referring to the image's source.
onSpotted function Function that fires when element is in view. Has three parameters given: 'element' (the element that is in view), the index of the element in the HTMLCollection and 'this' (referring to the class and gives access to the methods of Spotter).
onUnSpotted function Function that fires when element is out of view. Has three parameters given: 'element' (the element that is out of view), the index of the element in the HTMLCollection and 'this' (referring to the class and gives access to the methods of Spotter).

spotElements()
